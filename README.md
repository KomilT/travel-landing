## Getting Started

### Installation

1. Clone the repo:

```sh
git clone https://github.com/KomilT/travel-landing.git
```

2. Go to the project folder:

```sh
cd travel-landing
```

3. Install NPM packages:

```sh
npm install
```

## Usage

### Development server:

```sh
npm start
```

### Build:

```sh
npm run build
```

### Deploy:

```sh
npm run deploy
```

_(by default on GitHub Pages)._
